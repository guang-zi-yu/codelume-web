# CodeLume 官网

## 如何在本地编辑和预览该网站

建议使用 [pnpm](https://pnpm.io/) 作为本地开发的包管理器：

```bash
pnpm i
pnpm run dev
```

该项目要求 Node.js 为 `v18` 或更高版本。并且建议启用 corepack：

```bash
corepack enable
```
